## Resumo do estudo

Os dois produtos atendem o **mesmo mercado** (profissionais de TI/GRC no Brasil), mas capturam **momentos de carreira diferentes**:

- **START** — quem está **entrando ou destravando** em GRC TI: experiência indireta, salário R$ 5K–10K, dor de recolocação e insegurança técnica. *Vende entrada e segurança.*
- **PRO** — quem **já está dentro e quer subir**: experiência consolidada (68%), salário R$ 15K–20K, ambição de se tornar estratégico, consultor/auditor ou internacional. *Vende avanço e autoridade.*

> **Nota metodológica:** base pequena (START = 27 registros; PRO = 28). Os percentuais indicam tendência, não estatística definitiva.

## Privacidade

As bases de dados originais (CSV) contêm **dados pessoais de clientes** e, por isso, **não são versionadas neste repositório** (ver `.gitignore`), em conformidade com a LGPD. Apenas os dados agregados e anonimizados fazem parte do estudo publicado.
